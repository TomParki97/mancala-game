# Mancala 2

Static web app for playing Mancala (Kalah) with multiple AI levels. Built with Vite, React and TypeScript. Works on GitHub Pages under `/mancala-2/`.

## Development

```bash
npm ci
npm run dev
```

## Tests

```bash
npm test
npm run test:ui
```

## Build

```bash
npm run build
```
